# MONOCHROME — fashion e-commerce (портфолио-проект №17)

Дата: 2026-07-14
Статус: дизайн утверждён (вариант А — Frontend SPA + статичные данные)

## Цель

Портфолио-проект: сайт вымышленного магазина одежды «MONOCHROME» с кинематографичной
тёмной эстетикой. Весь визуал (видео и фото) генерируется в Higgsfield в едином
арт-дирекшне. Проект демонстрирует навыки: работа с тяжёлыми медиа, состояние
приложения, роутинг, анимации, адаптивность.

## Архитектура

- **SPA без бэкенда.** Товары — статичный JSON (`src/data/products.json`).
- **Корзина** — React Context + localStorage (переживает перезагрузку страницы).
- **Деплой** — статика, Vercel/Netlify (одна команда сборки `vite build`).

## Стек

- React 18 + Vite
- Tailwind CSS
- Framer Motion (появления при скролле, page-переходы, hover)
- React Router (страницы: /, /catalog, /product/:id, /lookbook; корзина — drawer)

## Страницы и компоненты

1. **Главная (`/`)**
   - Полноэкранное hero-видео (Higgsfield), поверх — название бренда и слоган, CTA в каталог.
   - Секция «Новая коллекция»: 4 карточки, при наведении статичное фото сменяется коротким видео/анимацией.
   - Лента lookbook-кадров (горизонтальный скролл или мозаика).
2. **Каталог (`/catalog`)**
   - Сетка 12–16 товаров, AI-фото в едином стиле.
   - Фильтры: категория, размер, цена; сортировка (цена ↑↓, новизна).
   - Состояние фильтров — в URL search params (шэрится ссылкой).
3. **Карточка товара (`/product/:id`)**
   - 2–3 ракурса фото, выбор размера, количество, «в корзину».
   - Блок «с этим носят» (2–4 товара).
4. **Lookbook (`/lookbook`)**
   - Редакторская страница: кампейн-видео + полноэкранные кадры, scroll-анимации.
5. **Корзина (drawer)**
   - Открывается с любой страницы, изменение количества, удаление, итоговая сумма,
     фейковый чекаут (форма → экран «спасибо за заказ», без реальной оплаты).

## Медиа-план (Higgsfield)

Единый арт-дирекшн: тёмная гамма (чёрный/графит/белый), студийный свет,
high-fashion стилистика, один «фотограф».

- 1 hero-видео (главная, loop, без звука)
- 2–3 lookbook-видео/анимации
- ~15–20 изображений: фото товаров (по 2–3 ракурса на ключевые товары) + кампейн-кадры
- Постеры (первый кадр) для всех видео — мгновенная отрисовка до загрузки видео

## Данные

`products.json`: id, name, category, price, sizes[], images[], описание, флаг isNew.
Категории: верхняя одежда, худи/свитшоты, футболки, брюки, аксессуары.

## Производительность и качество

- Lazy loading изображений (`loading="lazy"`), видео — `preload="metadata"` + poster.
- Адаптив: mobile-first, брейкпоинты Tailwind.
- Доступность: alt-тексты, фокус-состояния, семантическая разметка.

## Вне скоупа

- Реальная оплата, авторизация, бэкенд, админка, мультиязычность.

## Критерии готовности

- Все 4 страницы + корзина работают, адаптивны, наполнены AI-медиа.
- Корзина сохраняется после перезагрузки.
- `vite build` собирается без ошибок, сайт готов к деплою.
